# Prima

## Prerequisites

```sh
npm install -g pnpm
```

## Installation

1. Clone the repo

```sh
git clone https://github.com/hyfer/prima.git
```

2. Install dependencies

```sh
pnpm install
```

3. Copy .env.example to .env.local

```sh
cp .env.example .env.local
```

## Running the application

```sh
pnpm dev
```
